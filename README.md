# Weather Application Demo with Node.js

## Demo page

- https://yuki-weather-application.herokuapp.com/

## How to start

- install

```npm install```

- start

```npm run start```

- watch

```npm run dev```

※Node.js version 11以下

## API
- [weatherstock](https://weatherstack.com/)
- [mapbox](https://www.mapbox.com/)
- ※サンプルのアクセストークンを使用しています。
